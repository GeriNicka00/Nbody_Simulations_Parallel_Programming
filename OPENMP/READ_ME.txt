Running on BC4 instructions:

1) Compile using: python OMP_Setup.py build_ext -fi
2) Run using: python Multiple_Runs_OMP.py 2000 200 8 1

ATTENTION:
The second command runs the Multiple_Runs_OMP.py script
and submits many jobs for the chosen number of objects.
A folder will created with subfolders in which the summary
of performance has been saved. 

It is essentially an automated submission script.

Keep everything in the same directory as given.
